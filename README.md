## üîë How It Works
KanaShift uses advanced cryptography to transform your text. Here's a quick overview of the process:

- **Input Text**: Type in the text you want to mask.
- **Password**: Enter a password. This password is your key. Make sure to remember it.
- **Transformation**: The application uses a PBKDF2-hardened keystream to transmute your input into Japanese glyphs.
- **Output**: The masked output displays as garbled text, making it unreadable to anyone without the correct password.

## üí° Features
- **Password-bound**: Only you can unlock your masked text.
- **Reversible**: Easily decode your text when needed.
- **Strong Security**: Built with top-notch cryptography to protect your data.
- **User-friendly Interface**: Simple design for easy navigation.

## üìù Usage Instructions
1. Open KanaShift.
2. Enter the text you want to mask in the text box.
3. Type your password in the designated field.
4. Click on the "Mask" button. Your input will appear in the output section as Japanese glyphs.
5. To decode, enter the masked text and the same password, then click the "Unmask" button.
